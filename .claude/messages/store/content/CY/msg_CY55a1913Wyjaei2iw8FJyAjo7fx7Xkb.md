Here are the subagents available via the Task tool:

| Subagent | Purpose |
|----------|---------|
| **general-purpose** | Research complex questions, search for code, execute multi-step tasks |
| **statusline-setup** | Configure Claude Code status line settings |
| **Explore** | Fast codebase exploration - find files by patterns, search code for keywords, answer codebase questions |
| **Plan** | Software architect agent for designing implementation plans and strategies |
| **claude-code-guide** | Answer questions about Claude Code features, Claude Agent SDK, and Claude API |
| **archivist** | Meta-observer of data flows - sleuths through logs, git history, knowledge graphs, planning docs |
| **temporal-validator** | Data verification, staleness detection, temporal knowledge graph of validated facts |
| **agent-architect** | Catalogues and manages the fleet of agents, tracks relationships and gaps |
| **backend-architect** | Senior backend engineer perspective for architectural analysis and system reliability |
| **librarian** | Curator of external resources - URLs, citations, papers, datasets with provenance tracking |
| **systems-thinker** | Systems dynamics and complexity perspective - feedback loops, emergence, long-term behavior |
| **process-cartographer** | Maps processes, workflows, and systems - bottlenecks, learning systems, reward systems |

The **Explore** agent has three thoroughness levels: `quick`, `medium`, and `very thorough`.
