> ## Node Labels in Obsidian Graph View
> 
> ### Turn On Labels
> 
> In the graph view, click the **gear icon** (settings) and look for:
> 
> - **Text fade threshold** - Controls when labels appear based on **zoom level**
>   - Lower value = labels appear when zoomed out more
>   - Higher value = need to zoom in closer to see labels
>   - Set to **0** to always show labels
> 
> ### Labels Only for Larger Nodes?
> 
> **Not natively supported.** There's a [feature request on the Obsidian Forum](https://forum.obsidian.md/t/graph-text-fade-threshold-consider-node-size/6737) asking for exactly this - "text fade threshold should consider node size, so larger nodes show labels when zoomed out, like a map." But it hasn't been implemented.
> 
> ### Workarounds
> 
> | Option | How |
> |--------|-----|
> | **Zoom + fade threshold** | Set text fade threshold low, zoom to show only the labels you want |
> | **Juggl plugin** | [Advanced graph plugin](https://github.com/HEmile/juggl) with full CSS control over styling |
> | **Filter by links** | In graph settings, filter to show only notes with N+ links |
> 
> ### Quick Settings Path
> 
> 1. Open graph (`Ctrl+G`)
> 2. Click gear icon (top-left of graph)
> 3. Expand **Display**
> 4. Adjust **Text fade threshold** slider
> 
> ---
>

> - [Obsidian Graph View Help](https://help.obsidian.md/plugins/graph)
> - [Graph text fade threshold feature request](https://forum.obsidian.md/t/graph-text-fade-threshold-consider-node-size/6737)
