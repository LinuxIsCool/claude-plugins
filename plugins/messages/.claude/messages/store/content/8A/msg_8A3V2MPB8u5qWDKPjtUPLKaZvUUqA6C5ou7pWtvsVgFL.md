*We’ve released four new audio model snapshots* to make voice workflows faster, more performant and reliable:

* gpt-realtime-mini-2025-12-15
* gpt-audio-mini-2025-12-15
* gpt-4o-mini-transcribe-2025-12-15
* gpt-4o-mini-tts-2025-12-15

The new snapshots deliver lower error rates across real-world and noisy audio, fewer hallucinations, and improved instruction following and function calling. They’re also stronger in Chinese, Japanese, Indonesian, Hindi, Bengali, and Italian.

For scenarios where agent accuracy is critical, gpt-realtime remains our best performing audio model.

On *Tuesday, January 13* , the default versions of our voice models will automatically update to the *2025-12-15 snapshots*. You can start using them now by specifying the dated model IDs in your API requests. If you want to stay on the current versions after January 13th, explicitly set your model to:

* *gpt-realtime-mini-2025-10-16*
* *gpt-audio-mini-2025-10-16*
* *gpt-4o-mini-transcribe-2025-03-20*
* *gpt-4o-mini-tts-2025-03-20*

Learn more in our audio docs ( https://platform.openai.com/docs/guides/audio ) or try the new models in the OpenAI playground ( https://platform.openai.com/audio/realtime ).
